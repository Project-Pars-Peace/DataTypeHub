1. Modality Name

Mammography (you should cover the following modalities in this sheet; create more sheets if needed):

  • Conventional Mammography
  • Digital Mammography
  • Tomosynthesis (3D Mammography)
  • Contrast-enhanced Mammography (CEM)

2. Acronym

MG

3. Imaging Principle

Mammography acquires images by using low-dose X-rays to capture high-resolution images of breast tissue.

4. Data Type & File Format

4.1. Primary File Format(s)

  • DICOM (.dcm)
  • JPEG (.jpg, .jpeg) / JPEG 2000 (.jp2, .j2k)
  • TIFF (.tif, .tiff)
  • PNG (.png)
  • HDF5 (.h5)
  • NIfTI (.nii, .nii.gz)

4.2. Data Type

  • Grayscale (e.g., Standard 2D mammogram)
  • 3D Volumetric Imaging (Digital Breast Tomosynthesis - DBT)
  • Time-Series (in contrast-enhanced spectral mammography (CESM))
  • Functional Imaging (in Molecular Breast Imaging (MBI) and Contrast-enhanced Mammography (CEM))

4.3. Related Data Types (parent type)

Radiography

4.4. Standard Storage Format (which one in 4.1)

DICOM (.dcm)

5. Typical Resolution & Image Dimensions

5.1. Spatial Resolution

Ranging from 5 to 10 lp/mm, range from 1024×1024 to 4096×4096 pixels. (Reference)

5.2. Voxel Size / Pixel Spacing

Digital mammography detectors typically have pixel sizes between 50 and 100 µm. For example, a detector with a 100 µm pixel size would have a pixel spacing of 0.1 mm × 0.1 mm. (Reference)

5.3. Temporal Resolution (if applicable)

Not applicable.

6. Data Size (Storage Requirements)

6.1. Single Image Size

From 1 to 50 MB (Reference)

6.2. Typical Study Size

Four-View Screening Examination: from 30 MB to 200 MB Digital Breast Tomosynthesis (DBT): from < 800 MB to ~2.6 GB (Reference)

8. Common Clinical Applications

8.1. Primary Uses

Early Detection of Breast Cancer, Diagnostic Evaluation (Reference, Reference)

8.2. Specialized Uses

Digital Breast Tomosynthesis (DBT), Contrast-Enhanced Spectral Mammography, Photon-Counting Mammography (Reference)

9. Radiation & Safety Considerations

9.1. Ionizing Radiation

Yes

9.2. Radiation Dose (if applicable)

Approximately 0.2 mSv

9.3. Safety Concerns

Radiation-Induced Cancer, Pregnancy, Breast Implants, Post-Surgery or Radiotherapy (Reference)
